C程序设计专题
项目：迷宫游戏

如何运行本代码：
1）在windows环境中，运行devc++，使用默认的4.9版本gcc编译器。
2）进入Code目录，在devc++中打开main.dev文件，此时可以看到整个项目工程在文件中正常显示。
3）按F11编译运行程序。

游戏说明：
1）进入主界面，可以看到自定义地图、已保存的地图、默认地图、排行榜、退出、帮助等按钮。
2）自定义地图界面支持手动和随机生成地图，有一定限制条件和提示，支持保存地图。
3）已保存的地图界面支持加载地图并开始游戏。
4）游戏界面支持用户操作完成游戏和提示最短路。退出时将进入结算界面，结算界面支持显示自动求解、单步求解、展示所有路等功能。
5）默认地图界面提供一些默认地图可供游戏。
6）排行榜界面支持保存读取分数。
7）本游戏添加了简单的生命值（与时间相关）和钥匙、门算法。最短路和自动求解也支持含钥匙和门的机制。

注意：
如果您在读取已保存的地图或默认地图时出现读取失败，
可以打开代码的MapUtils.c文件的ReadMap函数，
将文件路径修改为注释掉的一条文件路径。
我们在多台设备的测试上注意到当前文件路径在大部分情况可以运行，
但在少部分电脑上需要切换成另一种。
